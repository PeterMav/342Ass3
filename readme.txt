Nothing special needed to run just execute the project
You can select from satellite or standard view for the map.
Once initial load and zoom to current location you can scroll around the map
Press and hold to add pin, only one pin can be on the map.
A button appears at the bottom once a pin has been placed, tap it
to go to the view that will show 5 historical images of the location
you have selected.

### Supported Devices
The app is universal it will run on both iPhone and iPad
iOS minimum requirement is 9.x

Thank you for enjoying this app!!
